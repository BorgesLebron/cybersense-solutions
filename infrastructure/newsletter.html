⚠ Threat Item 1

Fortinet SSL-VPN Zero-Day Actively Exploited by Volt Typhoon Targeting Critical Infrastructure

CVE-2024-21762 in Fortinet FortiOS allows unauthenticated remote code execution via crafted HTTP requests to the SSL-VPN web management interface. The Chinese state-sponsored group Volt Typhoon has been confirmed exploiting this vulnerability to pre-position inside U.S. energy and water sector networks — patch immediately or take affected devices offline.

⚠ Threat Item 2

Ivanti Connect Secure Emergency Directive: CISA Orders Federal Disconnection or Patch

CISA Emergency Directive 24-01 requires all federal civilian agencies to either patch Ivanti Connect Secure and Policy Secure appliances against CVE-2024-21887 / CVE-2023-46805 or disconnect them from agency networks within 48 hours. The chained vulnerability enables unauthenticated RCE and has been confirmed exploited in the wild against government targets.

Attackers modified workflow files in public and private repositories to extract environment variables containing cloud credentials, GitHub tokens, and code-signing keys during automated build runs. NIST's new special publication addresses the specific risk challenges of deploying large language models in SOC platforms, including prompt injection attack mitigation, AI hallucination impact in security alert triage, and supply chain verification requirements for AI model providers used in security tooling.

🎓 Training Byte

Detecting Credential Stuffing in Authentication Logs

Credential stuffing is characterized by high-volume authentication failures from geographically dispersed IPs against the same account — not brute force against multiple accounts. In your SIEM, build an alert: single username + more than 10 failed logins + more than 5 unique source IPs + within a 15-minute window. Success rates below 0.5% confirm stuffing, not guessing. Tag these for immediate SOC triage.

🎓 Training Byte

How to Identify a Pass-the-Hash Attack in Windows Event Logs

Pass-the-Hash attacks generate Event ID 4624 with Logon Type 3 (Network) using NTLM authentication, often followed immediately by privilege escalation events. Filter for Type 3 NTLM logons originating from non-standard workstation accounts during off-hours, particularly where the source and destination are both domain-joined systems.

🎓 Training Byte

Understanding Living-Off-the-Land Binaries (LOLBins) in Windows Defense

Living-off-the-land attacks use legitimate Windows binaries to execute malicious payloads, evading signature-based detection. The most commonly abused LOLBins include certutil.exe (for downloading payloads), mshta.exe (for executing HTA scripts), and regsvr32.exe (for DLL loading). Enable Sysmon and alert on these binaries executing with unusual parent processes or network connections.

🎓 Training Byte

Reading a Packet Capture: The First 10 Seconds of a Cobalt Strike Beacon

A newly executed Cobalt Strike beacon generates distinctive network artifacts in its first 10 seconds: a GET request to the C2 server with a specific URI pattern, an HTTP 200 response containing a base64-encoded payload in the response body, and subsequent check-in requests at regular jitter intervals. In Wireshark, filter for http.request.method == "GET" with a User-Agent field that appears legitimate but lacks standard browser fingerprinting headers.
